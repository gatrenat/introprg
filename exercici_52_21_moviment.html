Exercici 52_21. Reassignant responsabilitats

Context

  • Carpeta de lliurament: 52_21_moviment/

  • Continguts relacionats: Moviment de responsabilitat

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Considera si el mètode bonificacionsDeLloguer que hem creat per l'exercici anterior, correspon realment a la classe on es troba.

Aplica un moviment de mètode tot passant-lo a una classe més adequada. Les proves d'aquest exercici esperen que segueixis les convencions que vam fer servir amb la quantitat.

El codi resultant hauria de permetre que Client.informe() funcioni amb el següent codi:

 1public String informe() {
 2    double total = 0;
 3    int bonificacions = 0;
 4    String resultat = "Informe de lloguers del client " +
 5        getNom() +
 6        " (" + getNif() + ")\n";
 7    for (Lloguer lloguer: lloguers) {
 8        bonificacions += /* XXX crida al mètode mogut */;
 9
10        // composa els resultats d'aquest lloguer
11        resultat += "\t" +
12            lloguer.getVehicle().getMarca() +
13            " " +
14            lloguer.getVehicle().getModel() + ": " +
15            ( /* ??? */ * 30) + "€" + "\n";
16        total += /* ??? */ * 30;
17    }
18
19    // afegeix informació final
20    resultat += "Import a pagar: " + total + "€\n" +
21        "Punts guanyats: " + bonificacions + "\n";
22    return resultat;
23}

Nota: Ara eliminar la variable temporal bonificacions no serà tan directe com amb quantitat. De moment la mantindrem.
