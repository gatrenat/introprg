Exercici 36_02. Elimina taula CATEGORIES

Context

  • Carpeta de lliurament: 36_02_drop/

  • Continguts relacionats: El Zoo persistent

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Implementa el mètode Zoo.eliminaTaulaCategories() de manera que elimini la taula CATEGORIES si existeix, i no faci res en cas contrari.

Modifica el mètode Zoo.creaTaulaCategories() de manera que abans de crear la taula, la elimini si ja existia.

El resultat haurà de permetre que funcioni el següent codi:

 1import java.sql.SQLException;
 2public class UsaZoo {
 3    public static void main( String args[]) throws SQLException {
 4        Zoo zoo = new Zoo();
 5
 6        System.out.print("Primer connectem amb la base de dades: ");
 7        zoo.connecta();
 8        System.out.println("connectat");
 9
10        System.out.println("Eliminem la taula CATEGORIES per si existia");
11        zoo.eliminaTaulaCategories();
12
13        System.out.println("Creem la taula CATEGORIES (primer cop)");
14        zoo.creaTaulaCategories();
15
16        System.out.println("Eliminem la taula CATEGORIES");
17        zoo.eliminaTaulaCategories();
18
19        System.out.println("Creem la taula CATEGORIES (segon cop)");
20        zoo.creaTaulaCategories();
21
22        System.out.println("Creem la taula CATEGORIES (tercer cop)");
23        zoo.creaTaulaCategories();
24
25        System.out.print("Finalment tanquem la connexió amb la base de dades: ");
26        zoo.desconnecta();
27        System.out.println("desconnectat");
28    }
29}

Que haurà de generar la següent sortida:

$ java UsaZoo
Primer connectem amb la base de dades: connectat
Eliminem la taula CATEGORIES per si existia
Creem la taula CATEGORIES (primer cop)
Eliminem la taula CATEGORIES
Creem la taula CATEGORIES (segon cop)
Creem la taula CATEGORIES (tercer cop)
Finalment tanquem la connexió amb la base de dades: desconnectat
