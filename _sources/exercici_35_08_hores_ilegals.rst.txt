.. rubric:: Enunciat

Torna a considerar la classe ``Hora``.

La versió anterior estava fent servir un tipus d'excepció molt genèric. De
fet el més genèric: ``Exception``!

En aquest exercici faràs servir una excepció més concreta:
``IllegalArgumentException``.

Els missatges seran els mateixos que els corresponents a l'exercici
:doc:`anterior <exercici_35_07_hores_excepcionals>`:

* ``"hores fora de rang: «hores»"``
* ``"minuts fora de rang: «minuts»"``
* ``"segons fora de rang: «segons»"``

El següent codi ha de poder funcionar:

.. code-block:: java
    :linenos:

    public class UsaHora {
        public static void main(String[] args) {
            System.out.println("Intentarem crear una hora amb valors no vàlids");
            try {
                Hora hora = new Hora(-1, 1, 1);
            } catch (IllegalArgumentException e) {
                System.out.println("Argument il·legal!");
            }
        }
    }

L'execució generarà:

.. code-block:: console
    :emphasize-lines: 2-

    java UsaHora 
    Intentarem crear una hora amb valors no vàlids
    Argument il·legal!
