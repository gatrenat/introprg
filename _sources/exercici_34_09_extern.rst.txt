.. rubric:: Context

* Carpeta de lliurament: ``34_09_extern/``

* [✓] Exercici amb :ref:`autoavaluació<descripcio_exercicis_autoavaluacio>`

.. rubric:: Enunciat

Desenvolupa un programa a partir de la següent descripció en UML:

.. uml::
    :align: center

     @startuml
     hide circle
     skinparam monochrome true
     skinparam classAttributeIconSize 0


     class Ascensor {
        pis: int
     }

     class UsaAscensor {
        {static} main(String[])
     }

     UsaAscensor -left-> Ascensor 

     @enduml

En ser executat, obtindrem aquesta resposta:

.. code-block:: console
   :emphasize-lines: 2-

    $ java UsaAscensor
    L'ascensor creat des de fora està al pis -1

**Nota**: per passar les proves d'aquest exercici, caldrà que ``Ascensor``
no tingui ``main()``.
