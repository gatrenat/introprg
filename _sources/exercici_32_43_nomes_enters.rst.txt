.. rubric:: Enunciat

Desenvolupa un programa que a partir dels arguments que rebi per línia de
comandes, mostri el resultat de sumar els enters que rebi.

Per passar les proves, el teu programa haurà de desenvolupar els següents
mòduls dins del programa `SumaEnters`:

.. code-block:: java

    public static int quantsEnters(String[] valors)     // nombre d'enters a valors
    public static int[] filtraEnters(String[] valors)   // enters que hi ha a valors
    public static int sumaEnters(int[] valors)          // suma dels valors

En cas que no hi hagi cap enter, el resultat serà 0

Considera les simulacions:

.. code-block:: console
   :emphasize-lines: 2

    $ java SumaEnters a 1 2 -2 +4 5 sis
    10

.. code-block:: console
   :emphasize-lines: 2

    $ java SumaEnters un dos tres
    0

Pista: ``filtraEnters()`` haurà de crear un array d'enters. Per saber la
mida de l'array a crear li serà útil ``quantsEnters()``.
