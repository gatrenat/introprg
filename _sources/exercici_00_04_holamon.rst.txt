|exerciseicon__T| Exercici 00_04. Hola Món!
###########################################

.. rubric:: Context

* Continguts relacionats: :doc:`ritual_holamon`

* Com lliurar-lo: :ref:`instruccions <instruccions_git>`

* [✓] Exercici amb :ref:`autoavaluació<descripcio_exercicis_autoavaluacio>`

.. rubric:: Enunciat


En aquest exercici comprovaràs el funcionament de ``prgtest``.

Fixa't com la icona que representa l'exercici es mostra amb
la marca ✓. Això t'indica que l'exercici disposa de prova automàtica.
Anem a executar-la.

Si no ho has fet encara, actualitza el teu repositori per si hi ha canvis. Per
exemple, si has fet coses a un altre lloc o bé jo t'he afegit actualitzacions.

.. code-block:: console

    $ git pull

Ara, obre un terminal, inicialitza l'exercici, crea i ves a la carpeta
corresponent (recorda ``prgtest init 00_04``).

Ara guarda el fitxer ``HolaMon.java`` dins la carpeta de lliurament que
s'indica a l'inici d'aquest enunciat.

Pots comprovar que a la carpeta hi és el fitxer ``HolaMon.java`` des del
terminal:

.. code-block:: console
   :emphasize-lines: 1, 3

   $ ls
   HolaMon.java
   $ cat HolaMon.java
   /* Programa que saluda al món */
   public class HolaMon {
       public static void main(String[] args) {
           System.out.println("Hola Món!");
       }
   }

Un cop te n'has assegura't que el codi del teu programa hi és, toca
compilar:

.. code-block:: console

    $ javac HolaMon.java

Probablement no et donarà cap error i t'haurà generat el fitxer
``HolaMon.class``. Ho comprovem:

.. code-block:: console
   :emphasize-lines: 2

   $ ls
   HolaMon.class
   HolaMon.java

Fas la prova de que el teu programa fa el que s'espera:

.. code-block:: console
   :emphasize-lines: 1

    $ java HolaMon
    Hola Món!

Com que fa bona pinta, el guardes al repositori:

.. code-block:: console

    $ git add HolaMon.java
    $ git commit -am "finalitzat 00_04"

Fixa't que el missatge del ``commit`` indica que s'ha finalitzat
aquest exercici. Et recomano molt que quan registris els canvis prenguis
una mica de temps per descriure què has fet. Aquests missatges t'ajudaran
molt per trobar una versió en cas que tinguis algun problema en el futur.

Ara és el moment de comprovar que el programa fa l'esperat:

.. code-block:: console

    $ prgtest
    S'ha trobat un error en executar la prova 01


    Execució del programa
    =====================

    L'execució ha estat la següent:

    $ java HolaMon 

    Sortida esperada
    ================

    S'esperava la següent sortida del programa:

    1!  Hello World!

    Sortida trobada
    ===============

    La sortida que ha generat el programa ha estat:
    1!  Hola Món!

    Discrepància
    ============

    Les següents línies difereixen
    1   Hello World!
    1   Hola Món!

Ops! Què ha passat? Sembla que el nostre programa no passa una de les
proves!

Si ens fixe'm bé en el que ens diu ``prgtest`` veiem que la sortida
esperada era ``"Hello World!"`` mentre que la que ha generat el nostre
programa ha estat ``"Hola Món!"``.

Però el meu programa estava bé, profe. Si he copiat exactament el teu
codi!

D'acord. He fet una *miqueta* de trampa aquí i he demanat que la prova
requereixi que la sortida del programa estigui en anglès amb l'objectiu
que poguessis veure un error de ``prgtest``

Ara que ja saps quin és el problema, corregeix el codi, torna a fer una
prova manual per veure que tot estigui bé, registra el canvi i passa un
altre cop ``prgtest``:

.. code-block:: console

    $ javac HolaMon.java
    $ java HolaMon
    Hello World!
    $ git commit -am "00_04 canvi idioma de la sortida"
    $ prgtest
    El teu exercici passa totes les proves

Un cop saps que el teu programa passa les proves, pots decidir pujar-lo al
repositori remot perquè jo te'l pugui corregir, o esperar-te a haver fet
més exercicis.

Suposem que ja has acabat de treballar per avui i vols lliurar:

.. code-block:: console

   $ git push

Si tot ha anat bé, ja pots donar-ho per lliurat.
