Exercici 35_10. Les posicions del gat Renat

  • Continguts relacionats: Enumerats

  • Com lliurar-lo: instruccions

Enunciat

Considera el següent diagrama:

Implementa la classe GatRenat que ofereixi els elements que apareixen en el diagrama.

Atenció: alguns mòduls llencen una excepció si el valor que reben no és vàlid.

Fes servir la següent classe per provar la nova funcionalitat:

 1 public class UsaGatRenat {
 2      public static void main(String[] args){
 3          try {
 4              GatPosicio posicio = GatPosicio.fromString(args[0]);
 5              GatRenat renat = new GatRenat(posicio);
 6              System.out.println("El gat Renat està: " + renat.getPosicio());
 7         } catch (IllegalArgumentException e) {
 8             System.out.println(e);
 9         }
10      }
11 }

La sortida serà:

 $ java UsaGatRenat dret
 El gat Renat està: DRET
