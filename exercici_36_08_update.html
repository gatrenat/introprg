Exercici 36_08. Modificació d'animals

Context

  • Carpeta de lliurament: 36_08_update/

  • Continguts relacionats: El Zoo persistent

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Els animals a la vida real no solen canviar gaire de nom o de categoria, però les persones que introduïm les dades de vegades ens equivoquem.

Per aquesta raó, la classe Zoo oferirà la possibilitat de canviar la categoria d'un animal amb el mètode canviaCategoria()

El mètode canviaCategoria() rep un animal i una categoria, i en finalitzar, a la base de dades hi ha un registre corresponent a aquest animal amb la nova categoria.

El mètode pot generar SQLException.

Cal tenir present els següents casos:

  • cas l'animal no estigui a la base de dades

    En aquest cas, canviaCategoria() funcionarà com afegeixAnimal() tot reemplaçant abans la categoria original de l'animal per la nova.

  • cas que l'animal estigui a la base de dades.

    Si la nova categoria no es troba a la base de dades, caldrà afegir-la primer com ja hem fet a afegeixAnimal().

    Finalment, un cop tenim resolt el tema de la categoria, ja podem executar la sentència UPDATE corresponent.

El codi de prova a UsaZoo és:

 1import java.sql.SQLException;
 2import java.util.Arrays;
 3import java.util.List;
 4import java.util.LinkedList;
 5public class UsaZoo {
 6    public static void main(String[] args) throws SQLException {
 7        Zoo zoo = new Zoo();
 8
 9        System.out.print("Primer connectem amb la base de dades: ");
10        zoo.connecta();
11        System.out.println("connectat");
12
13        System.out.println();
14        System.out.println("Creem les taules");
15        zoo.creaTaulaAnimals();
16        System.out.println("Taules resultants: " + zoo.getNomTaules());
17
18        System.out.println();
19        System.out.println("Introduïm una categoria");
20        Categoria mamifer = new Categoria("mamífer");
21        zoo.afegeixCategoria(mamifer);
22        ZooUtils.mostraCategories(zoo.recuperaCategories());
23        //
24        // creem una llista d'animals amb algun malament classificat
25        Animal balena = new Animal("balena", new Categoria("peix"));
26        Animal tarantula = new Animal("taràntula", new Categoria("insecte"));
27        List<Animal> animals = Arrays.asList(
28            new Animal("pardal", new Categoria("ocell")),
29            new Animal("gat", mamifer),
30            new Animal("guppy", new Categoria("peix")),
31            balena,
32            tarantula
33            );
34
35        System.out.println();
36        System.out.println("Afegim uns quants animals");
37        for (Animal animal: animals) {
38            zoo.afegeixAnimal(animal);
39        }
40        System.out.println("A la base de dades, els animals són:");
41        ZooUtils.mostraAnimals(zoo.recuperaAnimals());
42        System.out.println("A la base de dades, les categories són:");
43        ZooUtils.mostraCategories(zoo.recuperaCategories());
44
45        System.out.println();
46        System.out.println("Corregim la categoria de la balena a una ja existent");
47        zoo.canviaCategoria(balena, mamifer);
48        System.out.println("A la base de dades, els animals són:");
49        ZooUtils.mostraAnimals(zoo.recuperaAnimals());
50        System.out.println("A la base de dades, les categories són:");
51        ZooUtils.mostraCategories(zoo.recuperaCategories());
52
53        System.out.println();
54        System.out.println("Modifiquem la categoria de la taràntula a una que no existeix");
55        zoo.canviaCategoria(tarantula, new Categoria("aràcnid"));
56        System.out.println("A la base de dades, els animals són:");
57        ZooUtils.mostraAnimals(zoo.recuperaAnimals());
58        System.out.println("A la base de dades, les categories són:");
59        ZooUtils.mostraCategories(zoo.recuperaCategories());
60
61        System.out.println();
62        System.out.println("Intentem modificar la categoria d'un animal que no existeix");
63        zoo.canviaCategoria(new Animal("cavall de mar", mamifer), new Categoria("peix"));
64        System.out.println("A la base de dades, els animals són:");
65        ZooUtils.mostraAnimals(zoo.recuperaAnimals());
66        System.out.println("A la base de dades, les categories són:");
67        ZooUtils.mostraCategories(zoo.recuperaCategories());
68
69        System.out.println();
70        System.out.print("Finalment tanquem la connexió amb la base de dades: ");
71        zoo.desconnecta();
72        System.out.println("desconnectat");
73    }
74}
